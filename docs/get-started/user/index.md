# User Guide

This guide covers everything you need to get started with Minitrino, from
installation to running your first cluster with modules.

## Topics

**[Installation and Upgrades](installation-and-upgrades)** - How to install
Minitrino via PyPI or from source, upgrade to newer versions, and configure
Docker contexts for different environments.

**[Workflow Examples](workflow-examples)** - Comprehensive guide to using
Minitrino, including provisioning clusters, working with modules, accessing the
UI and CLI, and customizing configurations.

**[CLI Reference](cli-reference)** - Complete command reference with all
options, flags, and parameters for every Minitrino command.

**[Environment Variables and Config](environment-variables-and-config)** - Learn
about Minitrino's configuration hierarchy, including command-line options, shell
variables, config files, and how to customize cluster behavior.

**[Troubleshooting](troubleshooting)** - Common issues and solutions for
debugging Minitrino clusters and Docker containers.

**[Reporting Bugs and Contributing](reporting-bugs-and-contributing)** - How to
file issues, contribute code, and participate in the Minitrino project.
