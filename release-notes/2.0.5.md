- The `lib_install` command is now `lib-install` to fall in line with the [GNU
  style
  guide](https://www.gnu.org/software/libc/manual/html_node/Argument-Syntax.html)
- The `SEP_LIC_PATH` environment variable is now `LIC_PATH`
